# ระบบเช็คชื่อด้วยใบหน้าพร้อมเว็บแอป (Face Recognition Attendance System) 📸

ระบบเช็คชื่อเข้าเรียนด้วยใบหน้าแบบอัตโนมัติ ประกอบด้วยโปรแกรมสแกนใบหน้า (Python OpenCV + face-recognition GUI) และเว็บแอปพลิเคชันจัดการข้อมูลสด (Node.js Express + HTML/CSS/JS) รองรับการใช้งานแบบออฟไลน์และการแสดงผลแบบเรียลไทม์

---

## 🛠️ โครงสร้างสถาปัตยกรรมระบบ (Architecture)
ระบบนี้สลัดจากการเชื่อมต่อ Google Sheets เดิมและทำงานแบบ **Local-First** โดยใช้ฐานข้อมูล **JSON** ในตัวเครื่องทั้งหมดเพื่อความเสถียรและทนทาน:

1. **เครื่องสแกนใบหน้า (Python App - `main.py`):**
   * ทำหน้าที่สแกนใบหน้าผ่านกล้อง USB/CSI, ตรวจจับ และจับคู่บุคคลผ่านโมเดลปัญญาประดิษฐ์ (AI)
   * ส่งคำร้องเช็คชื่อไปที่เว็บแอปพอร์ต `5000` แบบเรียลไทม์
   * หากเครือข่ายหลุด/เว็บล่ม จะเก็บประวัติไว้ในไฟล์ `offline_attendance.json` ทันที และซิงค์ย้อนหลังให้อัตโนมัติเมื่อเว็บออนไลน์
   * รัน HTTP Server ภายในตนเองที่พอร์ต `5001` เพื่อรับคิวอัปเดตโมเดล AI (Auto-Retrain) จากหน้าเว็บทันทีที่มีนักศึกษาลงทะเบียนใหม่

2. **เว็บแอปจัดการ (Node.js Web App - `web_app/`):**
   * บันทึกข้อมูลและประวัติการสแกนทั้งหมดลงไฟล์ **`web_app/db.json` (ฐานข้อมูลเดี่ยวแบบ JSON)**
   * หน้าเว็บแผงควบคุมสไตล์ **Dark Glassmorphism** แสดงผลสถิติและคิวการสแกนสดผ่านหน้าเว็บด้วยเสียงและการแจ้งเตือนสี (SSE - Server-Sent Events)
   * แท็บกำหนดเวลาคาบเรียน (Sessions): ปรับแต่งช่วงเวลาเปิด-ปิดสแกน แทนการใช้ Cooldown จับเวลารายชั่วโมงแบบเก่า
   * แท็บลงทะเบียนนักศึกษาใหม่: ใส่รหัส ชื่อ-นามสกุล และอัปโหลดไฟล์รูปภาพหน้าผ่านเว็บแอปได้เลย
   * แท็บสำรองข้อมูล: ดาวน์โหลดประวัติการเช็คชื่อทั้งหมดเป็นไฟล์ `.json`

## 📂 โครงสร้างการจัดเก็บข้อมูลแบบ JSON
เพื่อความปลอดภัยและความง่ายต่อการกู้คืน ข้อมูลทั้งหมดใช้ฟอร์แมต **JSON** ทั่วถึงกัน:
* **`faces/`**: โฟลเดอร์เก็บไฟล์ภาพถ่ายใบหน้าแยกโฟลเดอร์ตามรูปแบบ `<รหัสนักศึกษา>-<ชื่อ>`
* **`web_app/db.json`**: ฐานข้อมูลเก็บรายชื่อนักเรียน ประวัติการลงทะเบียน คาบเรียนเช็คชื่อ และประวัติการเช็คชื่อเข้าเรียนทั้งหมด
* **`offline_attendance.json`**: ไฟล์จัดเก็บคิวเช็คชื่อสำรองสำหรับตอนออฟไลน์ชั่วคราวที่เครื่องสแกน

## 🚀 วิธีเริ่มต้นใช้งาน (Quick Start)

### 1. ติดตั้ง Library & โมดูลที่จำเป็น
**ความต้องการของระบบ:** Python 3.10 ขึ้นไป และ Node.js v20 ขึ้นไป

*   **ติดตั้งฝั่ง Python (กล้องสแกน):**
    *   **สำหรับ Windows (เครื่องพัฒนา):** ไม่ต้องติดตั้ง `cmake` หรือคอมไพเลอร์เพิ่ม สามารถติดตั้ง `dlib` ได้ทันทีจากไฟล์สำเร็จรูป (.whl) ในโครงการ:
        ```bash
        pip install dlib-19.22.99-cp310-cp310-win_amd64.whl
        ```
    *   **สำหรับ Raspberry Pi 5 (หรือ Linux อื่นๆ):** ต้องติดตั้งตัวคอมไพเลอร์และ `cmake` เพื่อคอมไพล์โค้ด C++ ของ `dlib` ก่อนติดตั้ง:
        ```bash
        sudo apt-get update
        sudo apt-get install -y build-essential cmake g++ libx11-dev libatlas-base-dev
        pip install dlib
        ```
    *   จากนั้นติดตั้งโมดูลหลักตัวอื่นๆ:
        ```bash
        pip install opencv-python face-recognition requests pillow
        ```

*   **ติดตั้งฝั่ง Web App (เว็บจัดการ):**
    ```bash
    cd web_app
    npm install
    ```

---

### 2. วิธีการรันระบบเพื่อสแกนเช็คชื่อ (รันใช้งานปกติ/พัฒนาโปรเจกต์)
ต้องเปิดใช้งานทั้ง 2 ส่วนควบคู่กัน:

1.  **เปิดเว็บเซิร์ฟเวอร์ (Terminal 1):**
    ```bash
    cd web_app
    node server.js
    ```
    *จากนั้นสามารถเปิดเว็บแผงควบคุมหลักได้ที่ลิงก์: [http://localhost:5000](http://localhost:5000)*

2.  **เปิดโปรแกรมกล้องสแกนใบหน้า (Terminal 2):**
    ```bash
    python main.py
    ```

---

### 3. การรันระบบและเปิดใช้งานอัตโนมัติบน Raspberry Pi 5 (Autostart)
สำหรับเครื่องที่ติดตั้งเพื่อใช้งานจริงในตู้เช็คชื่อสแกนหน้า สามารถตั้งค่าให้ระบบรันขึ้นมาพร้อมกันได้ง่ายๆ:
*   **รันผ่านสคริปต์เดียว:** รันไฟล์ [run_local.sh](file:///D:/face_recognition/run_local.sh) เพื่อเปิดทั้ง Web App (Node.js), บราวเซอร์รายงานผล และกล้องสแกนหน้า (Python) พร้อมกันโดยอัตโนมัติ:
    ```bash
    ./run_local.sh
    ```
*   **การตั้งค่า Autostart เมื่อเปิดเครื่อง:** สามารถตั้งค่าให้ระบบเรียกใช้ [run_local.sh](file:///D:/face_recognition/run_local.sh) เมื่อเข้าสู่หน้าจอ Desktop โดยอัตโนมัติ
*   อ่านขั้นตอนทั้งหมดอย่างละเอียดได้ที่คู่มือ: [.agents/skills/face_recognition_pi5/SKILL.md](file:///D:/face_recognition/.agents/skills/face_recognition_pi5/SKILL.md)

## 💾 การกู้คืนและสำรองข้อมูลระบบ (Backup & Recovery)
* **การสำรองข้อมูล (Manual Backup):** เพียงทำการคัดลอกไฟล์ `web_app/db.json` และโฟลเดอร์ `faces/` เก็บไว้ ท่านจะสามารถกู้ข้อมูลทั้งหมดคืนมาได้ทันทีเมื่อระบบมีปัญหา
* **การดาวน์โหลดไฟล์สำรอง:** สามารถไปที่หน้าเว็บแอปแท็บ **"สำรองระบบ (Backup/Restore)"** แล้วกดปุ่มดาวน์โหลดไฟล์ข้อมูล `.json` เพื่อนำไปบันทึกเก็บไว้ได้โดยตรง
