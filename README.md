# Face Recognition System 📸

ระบบจดจำใบหน้าด้วย Python รองรับภาษาไทยและจำได้หลายคน

## วิธีติดตั้ง (Installation)
1. ลง Python 3.10
2. โหลดไฟล์ dlib .whl มาติดตั้ง 
    ref : https://github.com/datamagic2020/Install-dlib/blob/main/dlib-19.22.99-cp310-cp310-win_amd64.whl
3. พิมพ์คำสั่ง:
   pip install opencv-python face_recognition requests

## วิธีใช้
สร้าง โฟลเดอร์ faces เอารูปใส่ ***ถ้าอยากให้เป็นชื่อเรา ต้องเปลี่ยนชื่อไฟล์***
แล้วรันไฟล์ main.py
